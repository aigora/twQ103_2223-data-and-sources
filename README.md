# Data & Sources

Nuestro proyecto consiste en una aplicación que ofrece información sobre las distintas propiedades del agua de las fuentes de tres barrios de Madrid (Lavapiés, Salamanca y Chamberí), pudiendo así analizar la calidad del agua, ya sea potable o no. El programa está orientado a personas con diferentes intereses, académicos o didácticos, con contenidos asequibles a cualquier público.

## Objetivos del trabajo

*Establecer un menú de opciones que permita: 1. visualizar las instrucciones de la aplicación; 2. inicio de sesión: aportando nombre, apellidos, profesión y provincia; 3. verificación de datos, es decir, el programa muestra por pantalla los datos introducidos para que el usuario compruebe que son correctos; 4. inicio del programa, donde se abre el menú inicial con cuatro alternativas (1. consultar datos almacenados de fuentes; 2. promedio de propiedades de las fuentes; 3. valoración final; 4. salir). Por último, la opción 5. salida del programa.

*Se utilizarán 9 ficheros (archivo.txt), 8 de lectura: 3 son de tres barrios distintos con sus correspondientes fuentes; otros 4 con la información relativa a los coliformes, turbidez, pH y conductividad; y otro con las instrucciones. El último fichero se genera automáticamente cuando el usuario introduce sus datos personales.

*Describimos funciones para varias funcionalidades descritas a continuación: banner; inicio de sesión; imprimir datos del fichero de inicio de sesión, es decir, de los datos personales introducidos (generación de un fichero); imprimir datos de los ficheros; lectura del fichero instrucciones; verificación de datos personales; promedio encuestas; encuesta de valoración de la aplicación; media conductividad; contador conductividad; calidad conductividad; media turbidez; contador turbidez; calidad turbidez; media pH; contador pH; calidad pH; calcular coliformes; contador coliformes; calidad coliformes; menú promedio de los parámetros del agua; menú inicial.  

*Utilización de 3 vectores de estructuras: para guardar los datos del usuario, para las propiedades del agua de las fuentes, y para leer la cabecera de los ficheros.
